Direction

Vision

Long-term goal: build interconnected IP ecosystems where each product contributes to a larger structured narrative network — worlds that expand, reference each other, and evolve over decades.

Phase 1 — Foundation

Establish core engine technology, narrative graph systems, and the first flagship titles that define our visual and mechanical language.

Phase 2 — Network

Connect projects through shared lore, cross-title events, and a unified player identity that persists across games.

Phase 3 — Ecosystem

Open tools for community creators, modders, and partner studios to extend the universe with official narrative integration.

Phase 4 — Legacy

Build IP that outlives any single product — worlds players return to for years, with history shaped by millions of decisions.
